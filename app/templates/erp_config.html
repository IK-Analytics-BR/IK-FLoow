Informações sobre Integração com ERP
Requisitos para Integração
  • API REST disponível no sistema ERP
  • Credenciais de acesso com permissões adequadas
  • Mapeamento de campos entre os sistemas
  • Conexão de rede estável entre os sistemas
Tipos de Dados Sincronizados
  • Equipamentos: Informações básicas, localização, status
  • Estoque: Peças, quantidades, custos unitários
  • Ordens de Serviço: Detalhes, custos, status
  • Clientes: Informações básicas de contato
Importante: A sincronização pode substituir dados existentes. Certifique-se de fazer backup antes de sincronizar pela primeira vez.
