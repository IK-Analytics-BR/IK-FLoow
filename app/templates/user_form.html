Informações sobre Perfis e Permissões

O sistema possui dois perfis de usuário:

  • Administrador: Acesso completo ao sistema, incluindo gerenciamento de usuários e permissões.
  • Usuário: Acesso limitado às funcionalidades do sistema, de acordo com as permissões atribuídas.

As permissões controlam o acesso a funcionalidades específicas do sistema:

  • Usuários com perfil Administrador têm acesso a todas as funcionalidades, independentemente das permissões atribuídas.
  • Usuários com perfil Usuário só têm acesso às funcionalidades correspondentes às permissões atribuídas.

Para gerenciar permissões, acesse Permissões.

Recomendações de segurança:

  • Use senhas fortes com pelo menos 8 caracteres, incluindo letras maiúsculas, minúsculas, números e símbolos.
  • Não compartilhe credenciais de acesso com outros usuários.
  • Limite o número de usuários com perfil Administrador.
  • Atribua apenas as permissões necessárias para cada usuário.
