Informações sobre Integração com IoT
Formato de Dados MQTT

Os dispositivos devem publicar dados no formato JSON no tópico:

cmms/equipment/{device_id}/data

Exemplo de payload:

{
  "device_id": "dev123",
  "hour_meter": 1234.5,
  "sensors": {
    "temperature": 25.5,
    "vibration": 0.8,
    "pressure": 2.1
  },
  "alerts": [
    {
      "type": "high_temperature",
      "message": "Temperatura acima do normal",
      "priority": "high"
    }
  ]
}
Formato de Dados REST API

Para enviar dados via API REST, faça uma requisição POST para:

{{ url_for('integration.iot_api_receive_data', _external=True) }}

Inclua o cabeçalho X-API-Token com o token configurado.

O corpo da requisição deve seguir o mesmo formato JSON do MQTT.

Importante: Mantenha o token de API seguro e não o compartilhe.
